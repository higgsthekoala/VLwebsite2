Overview

This style guide documents the visual identity, design system, and component library for the Sonar Music website. It serves as a comprehensive reference for maintaining consistency across all digital touchpoints.

Design Philosophy

Our design philosophy centers on creating immersive, cinematic experiences that reflect the quality and artistry of our sound design work.

Technical Stack

Built with vanilla HTML, CSS, and TypeScript for maximum compatibility with Webflow and other platforms.

Accessibility

All components are designed with accessibility in mind, following WCAG 2.1 AA guidelines.

Accessibility

Color Contrast

All text must meet WCAG 2.1 AA standards with a minimum contrast ratio of 4.5:1 for normal text and 3:1 for large text.

Keyboard Navigation

All interactive elements must be accessible via keyboard navigation with visible focus indicators.

Screen Readers

Use semantic HTML and proper ARIA labels to ensure compatibility with screen readers.

Motion Sensitivity

Respect the prefers-reduced-motion setting and provide alternatives for users sensitive to motion.

Performance

Image Optimization

Use appropriate image formats (WebP, AVIF) and sizes. Implement lazy loading for below-the-fold images.

CSS Optimization

Minimize CSS, remove unused styles, and use efficient selectors. Prefer CSS over JavaScript for animations.

JavaScript Performance

Use modern JavaScript features, minimize bundle size, and implement code splitting where appropriate.
